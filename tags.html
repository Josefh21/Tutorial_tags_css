Vamos aprender um pouco sobre algumas tag's para HTML.


Primeira: HTML.

A tag HTML serve para especificar qual a linguagem que vai ser usada no programa. Podendo ser HTML, python, CSS, etc


Segunda: Header.

A tag Header é usada para fazer o cabeçalho do programa, fica fora do body e geralmente é usada para apresentar os títulos, subtítulos, etc.


Terceira: Head.

A tag Head fica fora do body também, quase sempre é a primeira tag a ser usada, dentro dela é colocado as informações que devem ser carregadas antes do próprio site, por exemplo, códigos de nome da guia, cor do site, língua (pt-br), etc.


Quarta: Body.

A tag body é usada para colocar o "corpo" do site, como os textos, imagens e etc.


Quinta: H.

A tag H é seguido por algum número entre 1 e 6, ela serve para escrever títulos, subtítulos, etc. Ex.:

H1

H2

H3

H4

H5
H6

Sexta: Section.

A tag Section é usada para separar o site por "setores" ou "sessões".


Sétima: Div.

A tag Div serve para criar um bloco para agrupar conteúdos.


Oitava: Footer.

A tag Footer serve para criar um "rodapé" do site, fica fora do body e geralmente é usada para dar os direitos autorais, contato, etc.


Nona: Hr.

A tag Hr é usada para criar uma linha de divisória, podendo ser usada por exemplo para separar o body do footer.


Décima: P.

A tag P é uma das mais usadas, ela serve para representar um parágrafo de texto.


Décima primeira: Br.

A tag Br é usada para fazer uma quebra de linha.


Décima segunda: Ul.

A tag Ul é usada para criar uma lista com marcadores.


Décima terceira: Li.

A tag Li é usada para criar itens dentro de alguma lista, geramente essa lista vem da tag anterior, Ul.


Décima quarta: Strong.

A tag Strong é usada para deixar o texto em negrito, assim como você está lendo.Ex.: HTML


Décima quinta: Img.

A tag Img serve para adicionar imagens.


Décima sexta: A.

A tag A é usada para criar links.


Décima sétima: Button.

A tag Button é usada para criar botões.


Décima oitava: Iframe.

A tag Iframa é usada para incorporar outro documento HTML no site.


Décima nona: Table.

A tag Table é usada para criar uma tabela.


Vigésima: Tr.

A tag Tr é usada para criar uma linha dentro de uma tabela.


Vigésima primeira: Td.

A tag Td é usada para definir uma célula de dados em uma tabela.


Vigésima segunda: Meta.

A tag Meta é usada para fornecer metadados ao documento HTML, fica dentro da tag head, e sempre é acompanhada de outras coisas como por exemplo, "utf-8", isso faz com que o HTML reconheça o documento como português, espanhol, etc.


Vigésima terceira: Form.

A tag form é usada para criar formulários.
